# TheF2E 2023 Mission2 - 總統即時開票全台地圖

![總統即時開票全台地圖](public/cover.png)

* [Demo 網址](https://lhchien33.github.io/TheF2E_2023_Mission2/)

## 作品說明

此網站目的為提供使用者查詢台灣地區歷年總統的選舉統計資料，目前僅有模擬資料

## 系統說明

**版本**

Node.js `v18.14.2`

**專案運行**

您可以按照以下步驟建置並運行專案：

```shell
npm install
npm run dev
```
